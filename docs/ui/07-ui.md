# 📱 07 - UI

> [!ABSTRACT] 💡 Em uma frase
> UI cross-play para mobile e PC com HUD simplificada — dual nomenclatura (técnico na interface, lore nos tooltips) e controles adaptados para cada plataforma.

---

## 🎮 Princípios de UI

1. **Dual nomenclatura:** Termos técnicos visíveis; nomes de lore em tooltips.
2. **Cross-play:** Mesma interface base, layouts distintos para mobile e PC.
3. **Legibilidade em combate:** HUD não sofre desaturação da Visão Verdadeira.
4. **Imersão:** Botões e labels usam terminologia de lore sempre que possível sem perder clareza.

---

## 📱 HUD Mobile

**Filosofia:** Simples, tocável, jogável com uma mão em situações básicas.

| Elemento | Posição | Tamanho mínimo |
|---|---|---|
| HP/SP bars | Canto superior esquerdo | Barras horizontais largas |
| Skill bar (4 slots) | Canto inferior direito | Botões circulares 60px+ |
| Minimapa | Canto superior direito | 120×120px, colapsável |
| Chat | Inferior esquerdo | Colapsável em ícone |
| Menu (inventário, stats) | Ícone central inferior | Toque abre painel lateral |
| Auto-ataque | Toggle inferior | ON por padrão em mobile |

**Auto-ataque:** Ativado por padrão em mobile. O personagem ataca automaticamente o inimigo mais próximo quando em combate — jogador foca em usar skills.

**Jogabilidade com 1 mão:** Movimentação por joystick virtual (esquerda); skills e ações com polegar direito. Possível em momentos de farm casual.

---

## 🖱️ HUD PC (Mouse + Teclado)

| Elemento | Posição | Controle |
|---|---|---|
| HP/SP bars | Canto superior esquerdo | — |
| Skill bar (10 slots) | Inferior central | Teclas 1–0; F1–F12 |
| Minimapa | Canto superior direito | Clique para expandir |
| Chat | Inferior esquerdo | Enter para digitar |
| Inventário | Tecla I | Janela flutuante |
| Status (stats) | Tecla C | Janela flutuante |
| Menu de sistema | ESC | Painel central |
| Auto-ataque | Tecla Tab | OFF por padrão em PC |

---

## 🌑 UI em Visão Verdadeira (Combate)

Quando a Visão Verdadeira está ativa:
- O cenário desatura (shader global) — **a HUD permanece com saturação normal** para legibilidade.
- Indicador sutil de VV ativa: ícone de "olho aberto" no canto da tela.
- Tooltips de inimigos mostram nome + tier (ex: *"Fagulha da Ira [Tier 1 — Desafio Justo]"*).
- HP bars de inimigos aparecem apenas ao atacar ou usar skill de revelação.

## 🖼️ Elementos de Interface Narrativos

| Elemento | Visual padrão | Lore |
|---|---|---|
| Tela de morte | Fade para luz branca; espaço de paz | Encontro com A Presença |
| Levelup | Onda de luz ao redor do personagem | "Seu Alinhamento Espiritual cresceu" |
| Expulsão de inimigo | Partículas de dissolução | "Expulsão confirmada" (não "Kill") |
| Abertura de Fissura | Distortion shader + som ambiente | Zona de evento dinâmico |
| Recebimento de missão | Livro se abre levemente | Encargo da Guerra |
