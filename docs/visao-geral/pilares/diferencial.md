# 💎 Diferencial

> [!ABSTRACT] 💡 Em uma frase
> O Advento não é um servidor de Ragnarok com skin cristã — é um jogo onde a teologia define cada mecânica, do loop de respawn ao sistema de progressão.

---

## 📝 O que nos Diferencia

### 1. Expulsão, não Morte
O MMORPG padrão tem "Kill Count". O Advento tem **Expulsões**:
- Nenhum inimigo "morre" — ele é expulso para "lugares áridos" (Mt 12:43).
- O respawn é teologicamente justificado: o mal retorna até a Parusia (Ap 20:10).
- A UI nunca mostra "mortes" — mostra "expulsões confirmadas" e "libertações".
- **Impacto:** O jogador sente que está combatendo algo eterno, não destruindo inimigos descartáveis.

### 2. A Parusia como Âncora de MMO
Todo MMORPG precisa de um "por que continuar":
- No Advento: porque a vitória definitiva (a Parusia) ainda não chegou.
- Cada dungeon limpa = resistência territorial.
- Cada MVP expulso = Principado temporariamente recuado.
- O endgame não é "zerar" — é vigiar até o Rei voltar.
- **Impacto:** Retenção orgânica baseada em escatologia, não em conteúdo artificial.

### 3. O Livro como Interface
A Bíblia não é item decorativo — é o que define o personagem como Player:
- Stats do personagem = revelados pelo Livro.
- Skills = manifestações de autoridade bíblica.
- Lente da Verdade = percepção espiritual via Santidade.
- **Impacto:** Cada melhoria de personagem é um ato de santificação, não de power gaming.

### 4. Dois Níveis de Profundidade
O Advento funciona em duas camadas simultâneas:
- **Superficial:** MMORPG divertido, classes balanceadas, farm recompensador, gráficos estilizados.
- **Profundo:** Cada mecânica tem âncora teológica; cada nome tem significado; cada boss é um Principado específico com implicação narrativa.
- **Impacto:** Casual e hardcore jogam o mesmo jogo em profundidades diferentes — sem excluir ninguém.

### 5. Consistência Total de Mundo
- Lore sólida do início ao fim baseada em proporção bíblica 2:1.
- Cada cidade tem nome dual (Sob o Véu / Nome de Revelação).
- Cada item tem nome de lore (não "Iron Sword" mas "Instrumento do Rei — Lâmina da Proclamação").
- A UI faz parte da imersão: barras de HP são "Integridade do Vaso".

## 🧠 Análise do Agente
> O diferencial é a "Camada de Propósito". Muitos MMORPGs falham porque o jogador sente que está em um ciclo vazio. No Advento, o loop de gameplay é amparado por uma escatologia clara que valida o tempo investido. O jogador não está matando tempo — está exercendo ministério enquanto aguarda a Parusia.
